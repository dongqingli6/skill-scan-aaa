VM Docker Evidence (Claude project — real strace + tcpdump + NOVA)

Two ways to use the Claude-side VM Docker pipeline:

A. Ingest already-generated evidence (offline)

If you already have a claude_output.txt + strace.log directory from a previous VM run, ASG will read it and feed it into the composite score.

B. Trigger live SSH run on VM — with Claude API

Requires asg/vm_config.json with VM host/user/credentials AND lanyiapi key. SCP skill to VM, start Claude CLI in Docker, pull artifacts. Costs ~$0.18 per run.

C. Paper-mode pure Docker (NO Claude / NO API)

Zero API cost. Runs .py/.sh inside Docker directly with strace + tcpdump, faithful to paper §3.4. Best for code-level attacks (e.g. reverse_shell_skill) where SKILL.md alone isn't enough.
